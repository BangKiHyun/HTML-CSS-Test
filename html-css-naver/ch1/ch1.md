4.  fieldset

   1. 그룹화

6. 리스트

   1.  ul 
      1. unordered list
      2. 정렬되지 않은 list

   3. ol
      1. ordered list
      2. 정렬된 list
      3. 번호로 정렬시켜 줌
   4. li
      1. ul, ol의 자식 태그
      2. list목록을 나열
